## 关于 Inkwell

**Inkwell** 是一个基于 [Astro](https://astro.build/) 的精致博客主题，专为追求品质与美感的写作者设计。

## 技术栈

- **框架**: Astro 5
- **样式**: Tailwind CSS v4
- **搜索**: Pagefind
- **评论**: Giscus
- **AI**: 支持 OpenAI / Anthropic / Gemini

## 快速开始

```bash
# 克隆主题
git clone https://github.com/your-github-user/your-repo-name.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
